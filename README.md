# 📈 Stock Forecast Dashboard

🚀 An interactive **Stock Market Analysis and Forecasting Dashboard** built using **R Shiny**.  
This dashboard allows users to visualize stock price trends, analyze technical indicators, and forecast future prices using **ARIMA** and **GARCH** models.

---

## 🌟 Project Overview

Financial markets generate large volumes of time-series data.  
This project demonstrates how **data science and time-series forecasting** techniques can be used to analyze stock trends and estimate future movements.

The dashboard provides:

✔ Real-time stock data visualization  
✔ Technical indicators for trend analysis  
✔ Time-series forecasting using ARIMA  
✔ Volatility modeling using GARCH  

---

## 📊 Dashboard Features

### 📉 Stock Price Trend
Visualize historical stock prices interactively.

### 📈 Moving Average Indicators
Includes:

- **MA20 (20-Day Moving Average)**
- **MA50 (50-Day Moving Average)**

These indicators help identify **trend direction**.

### 🔮 ARIMA Forecast
Predicts **future stock prices** based on historical data.

### ⚡ Volatility Analysis (GARCH)
Measures market **risk and volatility behavior**.

### 🎛 Interactive Controls
Users can:

- Select stock symbols
- Choose forecast duration
- Explore interactive charts
